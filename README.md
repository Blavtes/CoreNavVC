#### 一.为什么要开源？
开源是一种精神，时点软件感谢大家长期的支持与厚爱，无私开源奉献给大家，希望继续能支持我们：<br/>
> (1). 本框架是Charlin Feng项目必备的基础类框架。<br/>
> (2). 此框架已经在我自己的10多个正式项目中全面使用。<br/>
> (3). 多达4年经验精华积累，让框架更稳定好用实用<br/>
